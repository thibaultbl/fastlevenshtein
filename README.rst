Introduction
------------
Rust implementation of levensthein distance (https://en.wikipedia.org/wiki/Levenshtein_distance).

Installation
------------

::

   pip install fastlevensthein


Usage
------------

::

   from fastlevensthein import levensthein
   levensthein("string1", "string2")
   >>> 1
